### 第三條路（第 6-10 章）

| 角色 | 關係 | 功能 |
|---|---|---|
| **許芷涵** | 好友 · 保護者 | 末日前的死黨。末日後重逢結伴。主動犧牲代替語晴。 |
| **蕭亦晴** | 同班 · 保護者 | 三人結伴同行。先於芷涵被暴徒挑中致死。 |

**許芷涵 ↔ 蕭亦晴**：同班同學。在語晴身邊時默契地形成保護結構。

## 角色功能分類

### 對語晴的關係類型

| 類型 | 角色 |
|---|---|
| **保護者**（主動保護語晴） | 許芷涵、蕭亦晴、陳柏翰、黃培昇 |
| **加害者**（直接傷害語晴） | 黃志偉、劉彥廷、血狼軍閥（群體） |
| **教導者**（讓語晴學到生存技能） | 趙曉萍、鄭雅文 |
| **被保護者**（語晴想保護但失敗） | 葉沛妤 |
| **照顧者**（照顧語晴但非保護者） | 楊詩涵 |
| **託付者**（將重要東西交給語晴） | 吳若瑜 |
| **同路人**（短暫同行後分離） | 周心妍 |
| **碎裂者**（語晴旁觀其崩壞） | 方語彤、白若雪 |
| **情愫對象** | 黃培昇 |
