# ☄️ 星隕與蟲族入侵

> **字數目標：** 25萬字中文長篇連載  
> **尺度標籤：** R-18（強烈血腥、極度露骨性愛描寫、性暴力、黑暗心理）

---

## 📖 故事簡介

2026 年 1 月的火流星雨帶來了外星蟲族的六個月潛伏期。當末日無預警爆發，一名擁有傲人肉體卻對自身性感毫不自知、輕易引發周遭人類慾望與惡意的普通女高中生，必須在充滿變異混種的絕望廢土中掙扎求生。

在道德崩壞的世界中，性、暴力與權力結構被重新洗牌——這不是英雄拯救世界的故事。倖存者只能在夾縫中與低階變異體對抗，維持一種「蟲族懶得徹底殺光，人類也無力奪回地球」的詭異生存平衡。
