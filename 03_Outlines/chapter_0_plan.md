# 📝 新版第0章大綱草案：流星之夜（最終版）

> **🗺️ 劇情大綱規劃師產出** — 已採納五位專家審核建議，等待總導演最終確認
> **設計原則**：一個群體事件，十個角色同時登場。透過「對同一場流星雨的不同反應」展示每個人的性格。

## 🎭 本章登場角色與展示方式

> **原則**：角色不是被「描寫」的，是透過**在同一個事件中做了什麼、說了什麼**被記住的。

| 角色 | 層級 | 在流星雨中做了什麼（展示性格） |
|---|---|---|
| **林語晴** | 🔴 主角 | 雙手合十認真許願、把零食分給不認識的小朋友、完全沒注意到自己被多少人看 |
| **許芷涵** | 🔴 深度 | 嗓門最大、罵手機拍不清楚、把外套丟給語晴、嘴巴不饒人但位置永遠在語晴身邊 |
| **蕭亦晴** | 🔴 深度 | 安靜地坐在河堤邊緣，一個人看天空。語晴主動走過去坐她旁邊 |
| **陳柏翰** | 🔴 深度 | 眼睛不在天上——他在看語晴。散場時把外套搭在語晴腰後遮濕裙——語晴連他的名字都不確定 |
| **劉彥廷** | 🟡 中度 | 主動幫大家找好位置、分發觀星資料、完美得像在主持活動——但芷涵覺得怪怪的 |
| **方語彤** | 🟡 中度 | 帶了一大袋自製鳳梨酥來分給所有人，興奮地跟每個人聊天 |
| **周心妍** | 🟡 中度 | 坐在最不起眼的角落，但她的眼睛掃過了每一個人——她在記住一切 |
| **張雅婷** | 🟡 中度 | 看到大流星時大叫「哇靠超大！」——最吵的不是芷涵而是她 |
| **葉沛妤** | 🟢 一閃 | 低年級的小女生，抱著望遠鏡走過語晴面前差點跌倒，語晴扶了她一下 |
| **吳若瑜** | 🟢 一閃 | 沒有看天空——她低著頭在筆記本上寫東西，用文字記錄流星雨 |
| **爸爸** | 背景 | 在餐館送語晴出門時的互動 |
| **媽媽** | 背景 | 固定動作：撥碎髮。把外套搭在語晴肩上 |

## 📐 節拍 (Beats) 設計

### Beat 1：餐館打烊——出發前
**場景**：自家餐館，傍晚 6:00~7:00
**建議字數**：1,500 ~ 2,000 中文字
**任務**：建立家庭關係 + 傻白甜第一次展示

**內容**：

- **開場**：餐館快打烊了。油煙味、鍋鏟聲、暖黃日光燈。語晴站在櫃台後面邊收碗盤邊偷滑手機——芷涵在群組裡瘋狂刷訊息：「今晚河堤看流星雨！！已經好多人說要去了！！妳快來！！！」芷涵還轉發了一則新聞標題：「百年一遇！全球火流星潮今晚達到高峰，天文學家稱為世紀奇觀」

- **語晴的傻白甜展示**：
  - 語晴興奮地問爸媽能不能去。媽媽：「碗洗完了嗎？」語晴秒速衝去把剩下的碗洗完，動作太急打翻了一個杯子，嚇得蹲下來一邊撿碎片一邊跟杯子道歉：「對不起對不起……」爸爸無奈地笑：「妳跟杯子道什麼歉。」
  - 語晴換下圍裙準備出門。**肉體描寫的自然穿插**：她穿著的短袖小可愛在暖黃燈光下勾勒出明顯的輪廓。來吃晚飯的一個男客人視線飄了過來。媽媽注意到了——不動聲色地走過去，把語晴的薄外套從椅背上拿起來搭在她肩上。語晴茫然：「不會冷啦——」媽媽沒解釋，只是看了那個男客人一眼。
  - **爸爸的幽默**：「幫爸爸許個願，彩券中獎。」語晴認真地回：「好！我一定幫你許！」她是真的會幫爸爸許的那種人。

- **媽媽的固定動作**：語晴出門前，媽媽順手幫她把碎髮撥到耳後。語晴下意識地蹭了蹭媽媽的手掌。「早點回來。」「好——」
  - （這個動作會在後面的章節被反覆回想。每一次回想都更痛。）

### Beat 2：河堤群像——流星降臨
**場景**：社區河堤，晚上 7:30~9:30
**建議字數**：3,500 ~ 4,500 中文字
**任務**：十個角色同時展示性格 + 流星雨的壯麗 + 語晴傻白甜的多面展開
**節奏設計**：分「等待期」和「流星降臨」兩波——等待時嗨雜熱鬧，流星一來全場安靜，然後爆發歡呼

**內容**：

#### 第一波：等待期——角色登場

- **到場——人群與氣氛**：
  - 語晴小跑到河堤時，已經有不少人了。河堤旁的斜坡草地上鋪著野餐墊和塑膠椅，有同學、有附近的居民、有帶小孩來的家長。空氣中是冬夜的涼爽和河水的微腥味。
  - 芷涵在人群中揮手：「語——晴——這裡！」嗓門大到半條河堤都聽到了。

- **角色登場——透過互動而非描寫**：

  **【方語彤：暖陽】**
  - 語晴剛坐下，方語彤就端著一個大塑膠盒擠過來：「語晴——！我媽做的鳳梨酥，快吃快吃。」她的琥珀色杏眼笑成兩條縫，蓬蓬的及肩短捲髮在路燈下毛茸茸的。
  - 語晴接過鳳梨酥，咬了一口，眼睛亮了：「好好吃——！語彤妳媽也太厲害了吧！」那個表情認真到像是收到了什麼了不起的禮物。方語彤被她的反應逗樂了：「只是鳳梨酥而已啦。」但心裡暖暖的。
  - 方語彤把鳳梨酥分給在場的每個人——她記得每個人的名字，連隔壁班不太熟的同學都叫得出來。

  **【張雅婷：陽光】**
  - 斜坡上方傳來一個爽朗的聲音：「欸欸欸！你們有沒有帶望遠鏡！我忘了帶！」張雅婷從上面跑下來，高馬尾甩出乾淨的弧度，運動外套的拉鍊沒拉，裡面穿著排球社的練習服。
  - 她大剌剌地擠到語晴旁邊坐下：「林語晴對吧？上次體育課見過。」不等語晴回答就搶了一塊鳳梨酥：「讚！這誰做的？」

  **【劉彥廷：完美】**
  - 一個溫和的聲音從旁邊傳來：「各位同學，這邊有天文社整理的觀星方位資料，需要的可以來拿。」劉彥廷站在路燈下，銀框眼鏡、整齊的瀏海、微笑得體——一張列印好的觀星指南。
  - 語晴接過資料認真看，覺得「學長好用心喔」。
  - 芷涵湊過來瞄了一眼，小聲跟語晴說：「這個人也太積極了吧，又不是他辦的活動。」語晴：「人家熱心啊。」芷涵撇嘴，說不上來哪裡怪，就沒再說了。

  **【蕭亦晴：冰與沉默】**
  - 語晴注意到河堤邊緣——離人群稍遠的地方——坐著一個身影。蕭亦晴一個人坐在堤壩的石階上，黑色長直髮在夜風中微微飄動，膝蓋上攤著一本書——但她沒在看書，她在看天空。
  - 語晴猶豫了一下，拿了一塊鳳梨酥走過去：「亦晴！妳也來了啊，要不要吃鳳梨酥？」
  - 蕭亦晴看了她一眼——丹鳳眼在路燈下冷冷的，但語晴完全沒被嚇到。蕭亦晴接過鳳梨酥，輕聲說了一句：「……謝謝。」
  - 語晴開心地在她旁邊坐下了。蕭亦晴沒有拒絕，也沒有邀請——她只是繼續看天空。
  - 芷涵從遠處翻白眼：「那傢伙又不理人。」但她看到語晴坐在蕭亦晴旁邊有說有笑（其實只有語晴在說），也就沒去拉她回來。

  **【陳柏翰：沉默的注視】**
  - 斜坡後方的陰影處，陳柏翰坐在一棵樹下。他沒有加入任何群體——手裡拿著一罐沒打開的飲料，目光從暗處看向河堤邊的語晴。
  - 語晴和蕭亦晴坐在一起的畫面——語晴笑著說話、蕭亦晴安靜聽著——他看了很久。然後移開視線，低頭盯著自己的鞋尖。

#### 第二波：流星降臨——高潮

  - 九點左右，第一顆流星劃過天空。然後是第二顆、第三顆——很快，整片夜空被流星填滿了。
  - 人群爆發出驚嘆和歡呼。張雅婷大叫：「哇靠超大——！！！」芷涵舉著手機瘋狂拍：「靠北啊拍不到——手機爛透了——」方語彤拉著旁邊的人興奮地跳：「好漂亮好漂亮好漂亮——！」
  - **語晴**站在河堤上，淡藍色的眼睛映著漫天的流光。她雙手合十，閉上眼睛。芷涵湊過來：「妳許什麼？」語晴睜開眼笑了：「不能說，說了就不靈了。」（她許的是：「希望大家都平安。」）
  - **蕭亦晴**安靜地仰著頭。流星的光在她冰冷的丹鳳眼裡一閃一閃——同一場流星雨，在她的眼睛裡是更冷的光。她沒有許願——她不相信願望。但這片天空很美。她微微眨了眨眼睛——那是她極少見的、接近柔軟的表情。語晴在她旁邊輕聲說：「好漂亮喔，亦晴。」蕭亦晴沒回話。但語晴無意間轉頭的時候，看到她嘴邊有一小塊鳳梨酥碎屑。蕭亦晴注意到語晴的視線，伸手擦掉了——動作很快、很輕，像是不想被發現。
  - **陳柏翰**在暗處抬頭看天——流星的光映在他深棕色的眼睛裡。但只有一秒。他的視線又回到了語晴身上——她被流星照亮的側臉、被風吹起的長髮、合十的雙手。他把這個畫面刻進了記憶裡。
  - **劉彥廷**拿著手機拍了幾張照片，然後微笑著問身邊的人：「要不要我幫你們拍合照？」語氣永遠完美。
  - **周心妍**坐在角落。她沒有拍照、沒有許願、沒有歡呼。她只是安安靜靜地看著每一個人——看著他們笑、看著他們抬頭、看著他們對著天空許下不會實現的願望。她把這一切都記住了。

  **【葉沛妤：一閃而過】**
  - 一個低年級的小女生抱著天文社借的望遠鏡經過語晴面前，因為望遠鏡太重而走得搖搖晃晃。腳一滑差點摔倒——語晴反射性地伸手扶住了她：「小心！」小女生紅著臉道謝，匆匆跑走了。語晴連她的臉都沒記住。

  **【吳若瑜：一閃而過】**
  - 斜坡的另一邊，有個戴圓框眼鏡的短髮女生沒在看天——她低著頭，在一本厚厚的筆記本上飛快地寫著什麼。路燈的光照在她的筆記本上，密密麻麻的字。語晴路過時好奇地瞄了一眼，但看不清寫什麼。「她好認真喔。」語晴嘟囔了一句就走開了。

- **流星雨尾聲——語晴的傻白甜彙總展示**：
  - 流星雨開始減弱時，河堤旁有個小男孩哭了起來——他的氣球飛走了。語晴看到了，蹲下來笑著指著天空：「你看，你的氣球飛去跟流星一起玩了喔。」小男孩停止哭泣，抬頭看天。語晴趁機從口袋裡掏出最後一塊鳳梨酥：「流星說要送你這個當禮物。」
  - 旁邊的家長連忙道謝。語晴笑著擺手，跑回芷涵身邊。
  - 芷涵看了她一眼：「妳最後一塊也給人了。」
  - 語晴：「他在哭欸，我看不得小孩哭。」
  - 芷涵：「所以妳自己就不用吃了？」
  - 語晴摸摸肚子：「回家再吃泡麵好了。」露出一個傻傻的笑容。
  - （這就是林語晴。她永遠把自己排在最後面。）

### Beat 3：流星的尾巴——回家
**場景**：河堤 → 回家路上 → 語晴的房間，晚上 10:00~11:00
**建議字數**：1,000 ~ 1,500 中文字
**任務**：收束情緒 + 最後的純粹安寧

**內容**：

- **散場——陳柏翰的動作型告白**：
  - 流星雨漸漸減弱。人群開始散去。張雅婷大叫：「下次還有我還要來——！」方語彤跟每個人說再見，笑著揮手：「回去小心喔——」劉彥廷幫忙收拾地上的垃圾——即使不是他的。
  - 蕭亦晴站起來，拍了拍裙子上的草屑，沒有跟任何人道別就走了。但走出幾步的時候，她停了一下——回頭看了語晴一眼。語晴正在和芷涵打打鬧鬧。蕭亦晴移開視線，繼續走了。
  - **陳柏翰的迴避告白**：語晴站起來準備走，裙子後面有一小塊被河堤石階上的露水弄濕了——她完全沒注意到。陳柏翰從暗處走出來。他脫下自己的外套，走到語晴身後——猶豫了兩秒——然後輕輕地把外套搭在她腰後，剛好遮住濕掉的那塊。動作很輕，像怕驚動什麼。
  - 語晴轉頭：「欸？你的外套——」陳柏翰已經轉身走了。背影很快消失在人群裡。
  - 芷涵看了一眼：「那個男的是誰啊？」
  - 語晴歪頭想了想：「好像是我們班的……叫什麼來著……」
  - **她連他的名字都不確定。** 但他要穿著薄衣服在一月的夜晚走回家了。
  - 他把飲料罐丟進垃圾桶——那罐飲料自始至終沒有打開。

- **回家路上**：
  - 芷涵陪語晴走一段路。芷涵走在靠馬路的那一側——她從來不說為什麼，就是自然地走在外面。
  - 語晴哼著歌，抬頭看天上殘餘的流星。芷涵看著她的側臉——然後翻了個白眼：「笨死了。」但嘴角翹起來了。
  - 路邊有一個碎酒瓶散落在人行道上，語晴抬頭看天沒注意到。芷涵伸腳把它踢開了。語晴完全沒注意到。芷涵也沒提。
  - 芷涵到家後揮手：「明天見——」語晴：「明天見！今晚好開心！」

- **入睡前——最後的純粹安寧**：
  - 語晴回到家，媽媽留了一碗熱湯在餐桌上。她喝完湯，洗完澡，穿著寬鬆背心和短褲躺在床上。長髮半濕地散在枕頭上。
  - 她在班級群組發了一段話：「今天的流星雨超美的！！有人拍到好的照片嗎！」
  - 然後她抱著灰色兔子玩偶，看著窗外。天空安靜了——流星已經停了。
  - 她閉上眼睛，嘴角帶著微笑，沉沉睡去。
  - **最後一行**：窗外的夜空很安靜。所有的流星都已落盡。它們現在正靜靜地躺在深山裡、海底下、以及城市邊緣的荒地上——等待孵化。

## ⚠️ 寫作特別注意事項

1. **這一章沒有恐懼**：整章的情緒是溫暖→歡樂→壯麗→安寧。沒有任何不安的暗示。讀者在讀這一章的時候應該覺得「真美好」——直到後面的章節回頭看，才會意識到那些流星帶來了什麼。

2. **語晴的傻白甜不是靠描寫，是靠行為**：跟杯子道歉、真的會幫爸爸許願、把最後的鳳梨酥給哭的小孩、主動走向蕭亦晴、連周心妍都注意到了。讀者應該在讀完這章後發自內心地喜歡她——這樣後面的悲劇才會真正心碎。

3. **肉體描寫極精簡**：只有 Beat 1 媽媽搭外套的那一段自然穿插。本章的重點是「人」不是「肉體」。

4. **每個角色只需一個記憶點**：讀者不需要記住每個人的長相——只需要記住「方語彤是帶鳳梨酥的那個」「張雅婷是大叫哇靠的那個」「周心妍是坐在角落的那個」「陳柏翰是在暗處看語晴的那個」。一個行為 > 一段外貌描寫。

5. **最後一行是全書最殘忍的伏筆**：「它們在等待孵化」——讀者第一次讀不會在意這句話，但重讀時會毛骨悚然。
