# 🗺️ 第 0 章正式大綱
## 「序：流星之夜」

---

### 📌 章節定位
- **主線地圖座標**：序章（第 0 章）——整部故事的開場。
- **時間**：2026 年 1 月某個週五晚間（放學→餐館→流星雨）
- **地點**：第一海島區某座中型城市，公立高中→市區街道→家中小餐館→頂樓陽台
- **天氣/氛圍**：冬季但異常溫暖的傍晚（為語晴怕熱體質提供穿著合理性），入夜後天空被大量火流星劃過，新聞角落提及偏鄉失蹤案。
- **核心任務**：**讓讀者在末日降臨前先徹底愛上（並疼惜）這個女孩——透過「日常場景 + 無意識的肉體挑釁 + 傻白甜人格」的三位一體展演，建立她的致命吸引力與惹人憐愛的完整形象。**

---

### 🎬 敘事視角規範
- **主視角**：第三人稱有限視角，**緊貼林語晴**。所有內心獨白只走語晴一人。
- **允許的視角過渡**：在肉體描寫時，允許短暫切換至「旁觀者視角」（如客人的筷子停住、店員的視線黏附、機車騎士的回頭），但這些段落**僅描寫外部行為反應，不進入旁觀者的內心獨白**，用完即回到語晴視角。
- **嚴禁**：全知視角、敘述者劇透、任何暗示蟲族真相的旁白。

---

### 🎭 情緒曲線
```
輕鬆日常 → 微色氣的悶熱躁動 → 街道上一絲莫名的不適（漣漪）→ 溫暖的家庭歸屬感 → 流星雨的壯麗與微妙不安 → 餘韻中隱約的不祥預兆
```

#### Beat 1：放學——校園的最後一幕
**場景**：週五下午放學，教室→校門口
**建議字數**：2,500 ~ 3,000 中文字
**重點感官**：🔵 視覺主導 + 🟢 聽覺（語晴的聲音特質首次建立）

**內容**：
- 開篇即建立**視覺衝擊**：語晴穿著小可愛搭配短裙，外罩一件不拉拉鍊的薄外套，因為教室冷氣關掉後覺得悶熱，已經微微出汗。白皙的鎖骨與胸口泛著薄薄的水光。
- **無意識挑釁①——拉扯領口**：收拾書包時感覺悶熱，習慣性地勾住小可愛的領口往外拉扯透氣，動作自然到她毫無意識。身旁的男同學（陳柏翰，不點名暗戀身份，只用旁觀視角暗示）的視線短暫定格又移開。
- **聲音特質首秀**：語晴收好書包抬頭，看見芷涵在門口等她，立刻揚起笑臉喊了一聲「芷涵——」。她的嗓音清脆柔軟，尾音微微上揚，像一隻小鳥在枝頭啁啾，語速比一般人慢半拍，帶著一種天然的慵懶甜膩。這個聲音是讀者對她的第一個聽覺錨點。
- **許芷涵的互動加厚**：兩人走出校門時的日常對話——芷涵瞥了一眼語晴的穿著，嫌棄地說「一月欸妳穿這樣不冷嗎」，語晴理直氣壯地回「明明很熱啊」，芷涵翻白眼「只有妳覺得熱」。語晴傻笑著去拉芷涵的手臂，問今晚要不要一起看流星雨。芷涵說她要補習，語晴嘟著嘴巴遺憾地「蛤——」了一聲。這段對話建立兩人的親密度，為後續第 6 章的「末日重逢」預埋情感錨點。
- **傻白甜展現①**：小跑過去找芷涵時 F 罩杯在薄衣物下產生明顯的彈跳，她毫無自覺，只是因為晃動引起的胸口悶痛而皺了皺眉，用手臂環住胸前——這個「護胸小跑」的動作反而比正常跑步更具色情張力。
- **氛圍伏筆**：校門口電視牆滾動新聞——「今晚流星雨全島可見」，底部跑馬燈小字「中央山脈某登山隊已失聯第 9 天」，語晴只看見流星雨，開心地拉著芷涵說今晚要看。

#### Beat 2：回家路上——街道上的注視
**場景**：放學後的市區街道，從學校走到自家餐館的 15 分鐘路程
**建議字數**：2,000 ~ 2,500 中文字
**重點感官**：🔵 視覺 + 🟠 嗅覺（體香在微風中的擴散）

**內容**：
- **怕熱體質的全面展演**：一月的傍晚氣溫約 18~20 度，大多數路人穿著外套，語晴卻已經把薄外套脫下來搭在手臂上，只穿小可愛和短裙走在街上，白皙的肩膀、手臂與大腿大面積暴露。
- **無意識挑釁②——彎腰與舔唇**：路過便利商店幫餐館買調味料，蹲下取底層貨架時短裙繃到極限，勾勒出臀部的完整輪廓。站起來時因為蹲太久而微微搖晃，扶著貨架穩住自己，長髮從肩頭滑落。出了便利商店，嘴唇因為乾燥而無意識地伸出舌尖舔了一下——飽滿水潤的嘴唇被舌尖掠過後泛著一層薄薄的水光，這個完全出於生理本能的小動作，卻構成了一個與衣物無關的、純粹屬於五官層面的致命瞬間。
- **微妙的不適漣漪**：等紅燈時，旁邊的機車騎士摘下半罩式安全帽，視線從語晴的腿慢慢上移，在胸口停留了幾秒。語晴隱約感覺到那道目光的重量，下意識地把搭在手臂上的外套稍微往胸前攏了攏——但只是一閃而過的不自在，綠燈亮了她就把這件事忘得乾乾淨淨。這是她人生中「被注視」的無數次經歷之一，她從來沒有真正理解過那些目光的含義。
- **純淨體香**：走在街上時，微風吹來，她身上那股淡淡的牛奶香皂混合體溫的天然體香，在冬日微涼的空氣中格外明顯。路過的行人偶爾會不自覺地回頭。

#### Beat 3：餐館幫忙——汗水與布料的親密關係
**場景**：語晴家的平價小餐館，傍晚用餐尖峰時段
**建議字數**：3,500 ~ 4,000 中文字（本章核心 Beat，肉體性吸引力的集中爆發）
**重點感官**：🔴 觸覺（悶熱的體感、汗水的黏膩）+ 🟠 嗅覺（廚房油煙、體香的混合）+ 🟢 聽覺（鍋鏟聲、招呼客人的柔軟嗓音）
**光影設計**：小餐館的暖黃色日光燈。這種偏暖的黃光打在被汗水浸濕的白色小可愛上，會讓濕透的布料產生一種微微的半透明光暈效果——肌膚的粉白色與布料的濕潤白色在暖黃光源下融為一體，讓整個人看起來像是被裹在一層薄薄的蜜色光澤中。

**內容**：
- **場景建構**：小餐館的廚房蒸氣與炒鍋熱氣讓室溫偏高，油煙與飯菜的香氣混合在一起。語晴換上餐館的工作圍裙，但裡面仍然只穿著小可愛——圍裙只遮住前面，從側面和背面看，裸露的手臂、肩胛、腰側一覽無遺。
- **聲音特質的強化展現**：語晴穿梭在桌間招呼客人——「歡迎光臨」「請問幾位呢？」「小心燙喔」——她那清脆柔軟的嗓音在嘈雜的餐館裡格外清亮，語速慢慢的，每個字都帶著一種讓人不自覺放鬆的溫柔。即使只是在報菜名，那種像小動物細語般的音色也讓聽到的人耳朵微微一軟。
- **無意識挑釁③——透濕黏貼（核心場景）**：在廚房和外場之間來回端盤子，持續走動讓她開始大量出汗。白色小可愛逐漸被汗水浸濕，先是背部，然後是胸口——薄棉布浸透後緊緊黏貼在肌膚上，在暖黃色燈光的映照下，將 F 罩杯的完整輪廓、乳溝的深度、甚至內衣的顏色（淡粉色）都毫無保留地透出來。她只覺得不舒服，用手背擦了擦額頭的汗，完全沒意識到自己現在的模樣。
- **動態肉體描寫**：端著沉重的餐盤彎腰上菜時，胸部的重力垂墜在濕透的衣物下形成驚人的弧度；轉身時臀部的圓潤曲線在短裙下劃出流暢的弧線；小跑回廚房時，即使用一隻手臂半護著胸前，F 罩杯的晃動仍然明顯到讓坐在角落的男性客人筷子停在半空。
- **傻白甜展現②**：一桌帶小孩的客人小朋友打翻了湯，語晴蹲下來幫忙擦地板，一邊笑著安慰小朋友「沒關係沒關係」，淡藍色的眼睛彎成月牙——她蹲著的姿勢讓鎖骨以下的風景幾乎一覽無遺，但她的注意力完全在小朋友身上。
- **父母互動（建立溫暖→為後續毀滅鋪墊）**：爸爸從廚房遞出她最愛的滷肉飯，叫她先吃；媽媽經過時順手幫她把黏在臉頰上的長髮撥到耳後——語晴下意識地側頭蹭了蹭媽媽的手掌，像小動物依偎一樣自然。媽媽笑罵她「手油油的別蹭」，語晴縮了縮脖子吐舌頭傻笑。這個小動物般的依賴動作，是後續第 2 章「回家發現父母已死」的情感炸藥。語晴邊吃邊問爸爸今晚可不可以早點收工去看流星雨，爸爸笑著說好。
- **客人視角的暗線**：某桌獨自用餐的男性客人，從語晴進廚房到出來，視線始終跟著她移動——不做過多描寫，只用一句話暗示這具身體在狹小空間中的破壞力。

#### Beat 4：頂樓——流星雨之下
**場景**：餐館收工後，語晴爬上自家公寓頂樓看流星雨
**建議字數**：2,500 ~ 3,000 中文字
**重點感官**：🔵 視覺（流星雨的壯麗 + 沐浴後的肉體輪廓）+ 🟠 嗅覺（夜風中的沐浴後體香）
**光影設計**：流星的金色光芒間歇性地映在語晴白皙的肌膚上，每一道流星劃過時都會在她的臉頰與裸露的肩膀上鍍一層轉瞬即逝的暖金色光暈，與她淡藍色的瞳孔形成冷暖色的交錯渲染。

**內容**：
- **視覺轉換**：從悶熱喧鬧的餐館切換到開闊安靜的頂樓。夜風帶著一月的微涼。語晴洗過澡，穿著寬鬆的棉質背心和短褲，剛洗完的長髮還是半濕的，黑色髮絲貼在白皙的脖頸與肩膀上。
- **流星雨降臨**：天空中開始出現密集的火流星，光芒劃過夜空如同無數條金色的魚在黑色海洋中游過。語晴趴在頂樓矮牆上，雙手托腮仰望，淡藍色的眼瞳映著流星的金色尾跡，嘴巴微微張開——壯麗到讓她忘記了呼吸。
- **無意識挑釁④——沐浴後的致命輪廓**：趴在矮牆上的姿勢讓寬鬆背心的領口下垂，加上沒穿內衣的解放狀態，從側面看胸部的輪廓在背心布料下自然垂落，隨著她興奮地指著天空的動作微微搖晃。短褲很短，白皙的大腿幾乎完全暴露在月光下——但在她的世界裡，這只是「自家頂樓很自在」而已。
- **傻白甜展現③ + 小動物微動作**：她用手機拍流星（但手抖拍不好），嘟著飽滿的嘴唇賭氣地看著模糊的照片，然後放下手機乾脆用眼睛看。對著最亮的一顆流星認真地閉眼許願——許願時她不自覺地咬住了飽滿的下唇，雙手合十貼在胸前，眉頭微微皺起，認真得像在做一件全世界最重要的事。願望是「希望爸媽的餐館生意變好」——她永遠不會為自己許願。
- **體香在夜風中**：剛沐浴後的身體散發著比白天更濃郁的天然體香——沐浴乳的殘餘香氣混合著她本身的牛奶般的乾淨體味，在夜風中緩緩擴散。

#### Beat 5：尾聲——新聞裡的雜音與安睡
**場景**：頂樓→回到房間→入睡
**建議字數**：1,500 ~ 2,000 中文字
**重點感官**：🟢 聽覺主導（新聞聲、窗外的寂靜、窗玻璃的震動）

**內容**：
- **不祥的新聞伏筆**：回到房間，打開手機滑社群——同學們都在瘋傳流星雨照片。但她無意中滑到幾則不起眼的地方新聞：
  - 「南部某養殖場上千隻家畜離奇暴斃，場主失蹤」（對應蟲巢根系入侵養殖場）
  - 「北部某區地下水道工程因異常壅塞而無限期停工，施工人員報告管道內出現不明生物黏膜」（對應蟲巢根系入侵地下水道系統）
  
  她只覺得「好可怕喔」「好噁心喔」，然後就滑過去看同學的流星雨自拍了。
- **背景廣播**：隔壁房間爸爸看新聞的聲音隱約傳來——「…中央山脈登山步道已第三次封閉…搜救隊目前仍未尋獲…」語晴聽到但沒在意。
- **入睡畫面**：語晴躺在床上，長髮散落在枕頭上如一匹黑色絲綢，棉被只蓋到腰部（怕熱），背心微微捲起露出一小截白皙平坦的腰腹。她抱著一個舊娃娃，嘴唇微微嘟著，在流星雨的餘韻中安靜地睡著了——這是她人生中最後一個純粹安寧的夜晚。
- **末尾鉤子 (Hook)**：最後一段切到窗外的夜空——一顆比其他流星都更明亮、更緩慢、軌跡更低的火球，無聲地劃過城市的天際線，墜落在遠方中央山脈的方向。與之前金色/白色的普通流星不同，這顆火球的色澤偏向**暗沉的橘紅色**，拖著一道不自然的粗壯尾跡，如同一滴燃燒的暗血劃過夜幕。窗玻璃輕微地震了一下。語晴在睡夢中翻了個身，沒有醒。

### 🎯 設計考量

1. **「廢土戰損與色氣」濾鏡前置**：雖然第 0 章尚無廢土場景，但透過語晴在日常中的無意識肉體挑釁，預先建立讀者對這具身體的視覺記憶——讓讀者在後續末日章節中，能自動對照「這具曾經如此純淨的身體」正在被毀壞。
2. **四層遞進的肉體展演**：校園（微汗）→ 街道（暴露＋舔唇）→ 餐館（透濕黏貼——最高潮）→ 頂樓（沐浴後的無防備裸露感），每個場景的色氣濃度遞進上升，且挑釁手法涵蓋衣物類（拉領口、彎腰、透濕、無內衣）與非衣物類（舔唇），避免單一化。
3. **傻白甜的錨定**：每個肉體場景都搭配一個傻白甜行為，確保「肉體性感 + 心靈純真」的反差始終同步輸出。
4. **聲音特質的雙重錨定**：Beat 1（喊芷涵的清脆嗓音）與 Beat 3（招呼客人的柔軟語調）分別在校園與餐館場景建立語晴的聽覺形象。
5. **家庭溫暖 = 後續毀滅的燃料**：Beat 3 的父母互動越溫暖（蹭手掌的依賴動作），第 2 章「回家發現父母已死」的衝擊就越致命。
6. **新聞伏筆的「三重預埋」**：登山隊失蹤（偵察兵獵殺登山客）＋ 養殖場暴斃（地下根系入侵養殖場）＋ 地下水道壅塞（蟲巢根系入侵水道），三條線分別對應 `bug_biology.md` 的三個潛伏期活動，為第 1 章的詭異新聞累積提供更密集的預埋。
7. **色彩系統**：全章「暖白色調」（白皙肌膚＋淡粉內衣＋金色流星＋暖黃燈光），末尾以「暗橘紅色」的異常火球打破暖調，預告不祥。
8. **許芷涵的情感錨點**：Beat 1 的日常對話（吐槽穿太少＋流星雨邀約），為第 6 章末日後重逢時的「記得嗎那天晚上的流星雨」提供回憶素材。
